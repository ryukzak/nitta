NITTA.Model.Problems.Refactor

Description

Some times, CAD can not synthesis a target system because of a features of an algorithm and microarchitecture (too less process units, too many functions, too complicated algorithm).

In this case user can manually add some tweaks to the algorithm, but for that he should be an expert with deep understanding of NITTA project. Of course, it is not acceptable. This module defines type of that tweaks.
