VarValTime v x t => Testable (Multiplier v x t) v x Source #

The main purpose of this class is to generate autotests for PU. It allows to generate testbench for the PU according to its model and scheduled computational process. You can see tests in test/Spec.hs. Testbench contains:

  • The sequence of control signals that implement the already scheduled process.
  • The sequence of bus state checks in which we compare actual values with the results of the functional simulation.
