VarValTime v x t => TargetSystemComponent (Multiplier v x t) Source #

Usage of PU requires some artifacts of a synthesis process:

  • Hardware implementation, which depends from $sel:isMocked:Multiplier value:
  • Software (not needed for the multiplier because it does not have any configuration and is controlled from the network level).
  • Hardware instance in the upper structure element.
