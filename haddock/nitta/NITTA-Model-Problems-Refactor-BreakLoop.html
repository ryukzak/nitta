NITTA.Model.Problems.Refactor.BreakLoop

Description

BreakLoop example:

BreakLoop x o i

before:

l@( Loop (X x) (O o) (I i) )

after:

LoopEnd l (I i)
LoopBegin l (O o)

For more details: Loop
